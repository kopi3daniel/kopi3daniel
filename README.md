## 🚀 Mes projets principaux

### ⚡ Umeme Énergie
Plateforme de recharge de compteur prépayé avec vendeurs, clients, stock kWh, SMS et base de données locale.

### 🚦 Système de feux intelligents LoRa
Communication sans fil entre plusieurs carrefours avec ID, accusé de réception, stockage SD et API locale.

### 🐟 Samaki ya Kwetu
Système de monitoring pour pisciculture : température, pH, TDS, EC, gaz, volume d’eau et interface web locale.

### 🤖 IT-Robot-Astra
Projet de robot terrestre intelligent avec capteurs, caméra, LoRa, navigation semi-autonome et interface de contrôle.
